사용방법

( 이용안내 )

01.

Mobile,은 다운로드 창 에서
버튼 클릭 시 화면이 활성화 되며
Windows 용은 클릭후.
페이지 에서 다운로드 메뉴 클릭 후

설치 하면 해당 디바이스에
payda 설치가진행 됩니다.

02.

간편 계좌란?
기존의 중앙에서 자동으로 생성된
복잡한 형식이 아닌 본인의 계좌를
기억하기 쉽게 본인이 직접
만드는 형식입니다.

계좌 생성은 아래와 같이
빠르고 쉽게 생성 가능합니다

03.

payda를 사용하려면 은행을
등록 후 충전을 해야 사용 가능 합니다.
payda를 사용시 최초 1회 충전용
은행을 등록 해야 합니다.
충전 클릭 시 자동으로 이동 합니다.

충전하기 클릭시 등록한 은행 계좌에서
충전 신청 금액만큼 출금되며
payda는 충전 됩니다.

04.

송금은 payda 사용자 끼리 송금과
본인 명의 통장송금으로 나뉘며

payda 사용자간 송금은
제한이 없습니다.

05.

payda는 탈 중앙화 시스템으로 개인의 자산을 개인이 보관하는 장점이 있는 만큼 주의 해야 할 사항이 있습니다 주의 사항을 모두 숙지 후 서비스를 이용 해주시기 바랍니다

    warning
  • payda 서비스를 제공하는 중앙(회사) 서버에는 사용자의 아이디 비번이 저장되지 않습니다. 따라서 사용자가 비번을 잊어버린 경우 비밀번호정보 복구(찾기)가 절대 불가능 합니다.
    • warning
  • payda 디바이스(기기)이동, 복사, 보관은 백업을 통해서만 가능하며 비밀번호만 있다고 해서 다른 디바이스(기기)로 이동, 복사, 보관이 되지 않습니다.
    • warning
  • payda 간편 계좌 정보 (비번, 백업)의 분실, 삭제 및 유출 등 사용자의 부주의로 인하여 간편 계좌 지갑에 접근이 불가능한 경우 및 사용자 개인 자산의 피해는 완전히 사용자의 책임입니다.
    • warning
  • payda 즐겨 찾기 목록 또한 중앙(회사) 에 별도로 저장하지 않으며 프로그램을 삭제하거나 새로 설치 시 다시 등록해야 합니다. 백업은 수시로 받아야 하며 백업 파일 비밀번호를 잊어버린 경우 지갑 복원이 불가능합니다.
    • warning
  • 백업 파일은 최신 파일만 가지고 있어야 합니다.
    • warning
  • 백업파일이 여러 개일 경우 최신 파일이 아닌 다른 백업파일로 복구가 한 경우 최신파일로 다시 복구가 불가능 하여 최신 파일에 대한 자산의 피해가 발생 될 수 있으며 또한 다른 디바이스(기기)에서 복구한 뒤 로그인 을 하여 사용자 자산에 여러 문제가 발생할 수 있으니 최신 파일 1개만 가지고 있어야 합니다.
    • warning
  • 백업한 파일로 한번 복원한 경우 같은 파일로는 복원이 되지 않습니다. 복원 후 새로 백업을 진행해야 이후 다시 복원이 가능합니다.
